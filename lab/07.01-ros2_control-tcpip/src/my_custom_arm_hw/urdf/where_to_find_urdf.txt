# Serch URDF
```
# Root
find /  -name panda.urdf.xacro

E.g.
/opt/ros/humble/share/moveit_resources_panda_moveit_config/config/panda.urdf.xacro


# ros2_control.xacro
cp /opt/ros/humble/share/moveit_resources_panda_moveit_config/config/panda.ros2_control.xacro /ros2_ws/src/my_custom_arm_hw/urdf
```


# Convert xacro to urdf
```
# panda
ros2 run xacro xacro /opt/ros/humble/share/moveit_resources_panda_moveit_config/config/panda.urdf.xacro > /tmp/panda.urdf
```

# How to check your robot xacro
```
xacro my_robot.urdf.xacro > /tmp/out.urdf
```
